Creando repositorio remoto

Se accederá a la página de Github (o cualquier otra donde pueda guardar sus repositorios.

Una vez creado el repositorio, abrirá la terminal (o lo que corresponda, dependiendo su sistema operativo), y escribirá lo siguiente:

git init
git add .
git remote add origin https://github.com/kaibakev1984/kevinweb.git
git push -u origin master

pedirá el usuario y contraseña de su página donde alojará el repositorio.

Existen casos en los que se presentará el siguiente error:

To https://github.com/kaibakev1984/kevinweb.git
! [rejected] master -> master (fetch first)
error: fallo el push de algunas referencias a 'https://github.com/kaibakev1984/kevinweb.git'
ayuda: Actualizaciones fueron rechazadas porque el remoto contiene trabajo que
ayuda: no existe localmente. Esto es causado usualmente por otro repositorio
ayuda: realizando push a la misma ref. Quizás quiera integrar primero los cambios
ayuda: remotos (ej. 'git pull ...') antes de volver a hacer push.
ayuda: Vea 'Notes about fast-forwards0 en 'git push --help' para detalles.

Para resolver dicho problema, se escribirá lo siguiente por terminal:

git push origin +HEAD

Problemas para iniciar Color Picker en Sublime 3

Color Picker es un plugin bastante útil en Sublime 3, pues permite obtener el código hexadecimal del color deseado, haciendo "ctrl" + "shift" + "c"

En sistemas operativos como GNU/Linux, puede ocurrir que el mismo no funcione, puesto a que no se han instalado las dependencias necesarias.

Para hacerlo, se deberá abrir la terminal y escribir lo siguiente:

sudo apt-get install python-gtk2

Una vez instalado los paquetes, se deberá reiniciar Sublime Text, y este ýa funcionará.

En casos más extremos, se requerirá hacer lo siguiente:

sudo apt-get install libgtk2.0-dev

Hosting con Github Pages

Una forma de previsualizar tu proyecto, es usando Github Pages.

Para usarlo, es necesario que tengas abierto la terminal en la carpeta del proyecto. Necesitarás crear una rama llamada "gh-pages". Para ello, desde la terminal deberás escribir lo siguiente:

git checkout -b gh-pages
git add .
git commit -m "vamos a usar Github Web"
git push origin gh-pages

Para asegurarse de que se encuentra en la rama correspondiente, puede escribir lo siguiente desde la terminal:

git status

Una vez actualizado el repositorio, en el navegador deberá escribir lo siguiente:

nombredeusuario.github.io/nombreproyecto

En mi caso es:

kaibakev1984.github.io/kevinweb
